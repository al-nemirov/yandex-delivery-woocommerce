# Yandex Dostavka

Плагин интеграции с Яндекс Доставкой для WooCommerce.

## Установка

1. Скопируйте папку `yandex-dostavka` в `wp-content/plugins/`
2. Активируйте плагин в админке WordPress
3. Настройте методы доставки в WooCommerce → Настройки → Доставка

## Требования

- WordPress 5.0+
- WooCommerce 5.0+
- PHP 7.4+

## Сборка (для разработчиков)

Для блочного чекаута WooCommerce:

```bash
cd yandex-dostavka
npm install
npm run build
```
