Полная интеграция WooCommerce с Яндекс Доставкой: расчёт стоимости, выбор ПВЗ на карте, выгрузка заказов, печать ярлыков, синхронизация статусов, МойСклад.

== Description ==

Плагин для интеграции WooCommerce с Яндекс Доставкой через официальный B2B API.

= Способы доставки =

* **ПВЗ (самовывоз)** — клиент выбирает пункт выдачи на карте Яндекс
* **ПВЗ с наложенным платежом** — оплата при получении в пункте выдачи
* **Курьер до двери** — доставка по адресу клиента
* **Курьер с наложенным платежом** — оплата курьеру при получении

= Расчёт стоимости =

* Динамический расчёт через API Яндекс Доставки (вес, габариты, маршрут)
* Фиксированная наценка в рублях поверх цены API
* Процентная наценка поверх цены API
* Фоллбек на фиксированную стоимость при ошибке API
* Отображение срока доставки в днях
* Фильтрация по мин/макс весу отправления

= Выбор ПВЗ на чекауте =

* Интерактивная карта Яндекс с кластеризацией точек
* Список ПВЗ с адресами и графиком работы
* Поиск по адресу внутри модалки
* Сохранение выбора в cookies для устойчивости при обновлении чекаута

= Подсказки адресов (Dadata) =

* Автоподсказки города при вводе
* Автоподсказки адреса для курьерской доставки
* Настройка API-ключа Dadata в отдельном меню

= Выгрузка заказов =

* Создание заявки на доставку в Яндекс через API
* Авто-отправка при переходе в выбранный статус
* Ручная отправка кнопкой из карточки заказа
* Повторная отправка при ошибке (retry)
* Подтверждение заказа в API
* Настраиваемый префикс номера заказа

= Печать =

* Генерация и скачивание ярлыков (PDF)
* Генерация и скачивание актов передачи
* Авто-генерация акта после выгрузки заказа

= Синхронизация статусов =

* Крон-задача каждые 2 часа — опрос API Яндекс Доставки
* Автоматическое обновление статуса WooCommerce-заказа
* Авто-завершение заказа при вручении клиенту
* Полная история статусов в карточке заказа

= Интеграция с МойСклад =

* Автоматическая выгрузка заказов в МойСклад при смене статуса
* Ручная выгрузка кнопкой из карточки заказа
* Создание контрагентов и заказов покупателей
* Настройка организации и товара по умолчанию

= Администрирование =

* Метабокс в карточке заказа: трекинг, ярлыки, акты, ошибки
* Настройка пункта приёма с автокомплитом из API (12000+ точек)
* Проверка валидности API-токена
* Тестовый режим (скрывает методы доставки, не затрагивает боевые заказы)
* Совместимость с HPOS (High-Performance Order Storage)

= Чекаут =

* Поддержка классического и блочного чекаута WooCommerce
* Валидация: обязательный выбор ПВЗ для самовывоза
* Автозаполнение billing-полей для ПВЗ (адрес, индекс)
* Кастомные CSS-стили для чекаута

= Вес и габариты =

* Автоматический расчёт из свойств товаров
* Конвертация единиц (кг/г, см/м/мм)
* Габариты по умолчанию: к каждому товару или ко всему отправлению
* Расчёт объёма из суммы товаров

== Installation ==

1. Загрузите папку `yandex-dostavka` в `wp-content/plugins/`
2. Активируйте плагин через меню «Плагины»
3. Перейдите в **WooCommerce → ЯД — Настройки API** — введите ключ Dadata и Яндекс Карт
4. Перейдите в **WooCommerce → Доставка → Зоны** → добавьте метод «Яндекс Доставка — ПВЗ»
5. Введите OAuth-токен из личного кабинета Яндекс Доставки
6. Выберите пункт приёма заказов (начните вводить город)
7. Настройте наценки и автоматическую отправку

== Frequently Asked Questions ==

= Где взять OAuth-токен? =

В личном кабинете Яндекс Доставки → раздел «Интеграция» → API-токен.

= Где взять ключ Dadata? =

Зарегистрируйтесь на dadata.ru → Личный кабинет → API-ключи.

= Как работает наложенный платёж? =

Добавьте метод «Яндекс Доставка — ПВЗ (наложенный платёж)». Клиент оплачивает при получении. Сумма = товары + доставка.

= Как настроить МойСклад? =

WooCommerce → МойСклад Яндекс — введите логин/пароль, UUID организации и товара по умолчанию.
